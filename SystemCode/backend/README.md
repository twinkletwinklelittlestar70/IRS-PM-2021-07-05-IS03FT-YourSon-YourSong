# install

1. Install python

2. New a python env version=3.6

3. install dependencies

```
cd backend

pip install -r requirements.txt
```

4. Run the service

```
// for bash
export FLASK_APP=app
flask run

// for cmd
set FLASK_APP=hello
flask run

// for powershell
$env:FLASK_APP = "hello"
flask run
```
